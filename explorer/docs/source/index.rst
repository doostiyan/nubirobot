Welcome to Nobitex Explorer v1.0.0!
===================================

Overview
--------
   The Nobitex Explorer project is an integrated Django system that provides exploring services across various blockchain networks. This software uses the blockchain module developed by the Nobitex blockchain team and provides its services in the form of REST API. It is now possible to explore transaction details, wallet balance and wallet transactions for more than 20 currencies which include: BTC, ETH, DOGE, ADA, LTC, BCH, BSC, DOT, EOS, TRX, XLM, XRP, BNB, USDT, ETC, UNI, AAVE, LINK, DAI, GRT, SHIB, MATIC.
